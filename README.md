# GitBuddyBot 🤖

**GitBuddyBot** — это телеграм-бот, который помогает новичкам в программировании 
(и старичкам с провалами в памяти) быстро вспоминать команды Git с пояснениями, примерами и дружелюбным интерфейсом.

## 🧼 Стиль кода

Проект использует линтеры для проверки и автоформатирования:

- `black` — автоформатирование
- `flake8` — проверка PEP8

Для запуска проверки:
```bash
flake8 .
```
Для автоформатирования:

```bash
black .
```
